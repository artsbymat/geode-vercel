Geode supports embedding youtube, tweet, and markdown files

```markdown
![youtube](https://www.youtube.com/watch?v=446E-r0rXHI&t=1s)
![tweet](https://x.com/X/status/2002137003916472449?s=20)
![[markdown file.md]]
```
