# Whats is Geode?

> [!info]
> Geode is a static site generator written in Go. Geode is alternative obsidian publish like [quartz](https://quartz.jzhao.xyz/)

Why i build Geode? Because i want to learn Go and build something useful.

# Main Features

- Markdown syntax with obsidian extension
- Wikilink shortest path / absolute path
- Full text search
- Graph
- Hot reload
- Customize themes
- Latex
- Mermaid
